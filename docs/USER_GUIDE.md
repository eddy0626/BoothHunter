## 1. 설치 방법

### Windows 설치

빌드 결과물 중 하나를 선택하여 설치합니다.

| 설치 파일                         | 설명                      |
| --------------------------------- | ------------------------- |
| `BoothHunter_0.3.0_x64-setup.exe` | NSIS 설치 프로그램 (권장) |
| `BoothHunter_0.3.0_x64_en-US.msi` | MSI 설치 프로그램         |

1. `BoothHunter_0.3.0_x64-setup.exe`를 더블클릭합니다.
2. Windows SmartScreen 경고가 나타나면 **"추가 정보"** > **"실행"**을 클릭합니다.
3. 설치가 완료되면 바탕화면 또는 시작 메뉴에서 **BoothHunter**를 실행합니다.

### Linux 설치

| 설치 파일                               | 설명                           |
| --------------------------------------- | ------------------------------ |
| `BoothHunter_0.3.0_amd64.deb`          | DEB 패키지 (Ubuntu/Debian)     |
| `BoothHunter_0.3.0_x86_64.rpm`         | RPM 패키지 (Fedora/RHEL)      |
| `BoothHunter_0.3.0_amd64.AppImage`     | AppImage (portable, 설치 불요) |

### 시스템 요구사항

- **Windows**: Windows 10 (64-bit) 이상, WebView2 필요 (Windows 10+ 기본 포함)
- **Linux**: WebKitGTK 4.1+ 필요
- **인터넷 연결**: Booth.pm 검색에 필요

## 2. 앱 화면 구성

앱은 **좌측 사이드바**와 **메인 콘텐츠 영역**으로 구성됩니다.

### 사이드바

| 메뉴                | 기능                           |
| ------------------- | ------------------------------ |
| **검색**            | Booth.pm 상품 검색 (메인 화면) |
| **즐겨찾기**        | 저장한 즐겨찾기 목록           |
| **통계**            | 사용 통계 대시보드             |
| **VRChat 카테고리** | 카테고리별 빠른 필터링         |
| **언어 선택**       | 한국어/영어 UI 전환            |

> 창 크기가 1024px 미만일 때 사이드바가 아이콘 모드로 자동 축소됩니다. 아이콘에 마우스를 올리면 툴팁이 표시됩니다.

### VRChat 카테고리 목록

사이드바 하단의 VRChat 섹션에서 카테고리를 클릭하면 해당 카테고리로 필터링된 검색을 수행합니다.

- 3D 캐릭터, 3D 의상, 3D 소품, 3D 텍스처
- 3D 모델 (기타), 3D 장식품, 3D 환경/월드
- 3D 모션/애니메이션, 3D 툴/시스템, VRoid

## 3. 검색 기능

### 기본 검색

1. 상단 검색창에 키워드를 입력합니다.
2. **검색** 버튼을 클릭하거나 `Enter`를 누릅니다.
3. 검색 결과가 카드 형태로 표시됩니다 (페이지당 24개).

<!-- [스크린샷: 검색 결과 화면 — 카드 그리드] -->

### 한국어 → 일본어 자동 변환

한국어로 입력하면 자동으로 일본어 변환 제안이 드롭다운에 표시됩니다.

**지원 변환 유형:**

| 유형              | 예시                                 |
| ----------------- | ------------------------------------ |
| **아바타 이름**   | 키프펠 → キプフェル, 루루네 → ルルネ |
| **의상 이름**     | 원피스 → ワンピース, 치마 → スカート |
| **VRC 용어**      | 대응 → 対応, 전용 → 専用             |
| **카타카나 변환** | 임의의 한글 → 가타카나 자동 변환     |

드롭다운에서 원하는 변환 결과를 클릭하면 해당 일본어로 검색됩니다.

<!-- [스크린샷: 검색 제안 드롭다운 — "키프펠" 입력 시] -->

### 영어 → 일본어 자동 변환

영어로 입력해도 일본어 변환 제안이 표시됩니다.

| 유형              | 예시                                        |
| ----------------- | ------------------------------------------- |
| **아바타 이름**   | kipfel → キプフェル, rurune → ルルネ        |
| **의상 이름**     | dress → ワンピース, skirt → スカート        |
| **VRC 용어**      | compatible → 対応, exclusive → 専用         |
| **접두어 매칭**   | "sh" 입력 시 shirt(シャツ), shoes(シューズ) 등 후보 표시 |

한국어 변환과 동일하게, 드롭다운에서 원하는 결과를 클릭하면 해당 일본어로 검색됩니다.

### 인기 아바타 빠른 검색

검색창 아래에 인기 아바타 목록이 표시됩니다.

- **클릭**: `{아바타이름} 対応` (대응 의상 검색)
- **우클릭**: `{아바타이름}` (전체 검색)

<!-- [스크린샷: 인기 아바타 버튼 목록] -->

### 필터 기능

검색 후 나타나는 필터 바에서 결과를 세부적으로 조절할 수 있습니다.

#### 정렬

| 옵션        | 설명                  |
| ----------- | --------------------- |
| 최신순      | 최근 등록된 상품 우선 |
| 인기순      | 인기도(판매량) 기준   |
| 가격 낮은순 | 저가부터 표시         |
| 가격 높은순 | 고가부터 표시         |

#### 무료만 보기

**"무료만"** 체크박스를 활성화하면 무료 상품만 표시됩니다.

#### 가격 범위

**필터** 버튼 클릭 후 가격 범위를 입력하고 **적용**을 누릅니다.

- 최소: `0` ~ `999,999,999`
- 최대: `0` ~ `999,999,999`

#### 좋아요(Wish) 필터

상품의 좋아요 수 기준으로 필터링합니다.

1. 필터 패널에서 **"좋아요 최소"** 값을 입력합니다.
2. **적용**을 클릭합니다.
3. 설정한 수 이상의 좋아요를 받은 상품만 표시됩니다.

> 좋아요 데이터는 검색 후 백그라운드에서 자동으로 불러옵니다. "로딩 중..." 표시가 사라진 후 필터가 정확히 동작합니다.

#### 필터 초기화

활성 필터가 있을 때 **"초기화"** 버튼을 클릭하면 모든 필터가 해제됩니다.

<!-- [스크린샷: 필터 패널 펼침 — 가격/좋아요 필터 입력 상태] -->

### 페이지 이동

검색 결과 하단의 페이지 번호를 클릭하여 페이지를 이동합니다.

## 4. 번역 기능

상품명과 설명을 일본어에서 한국어로 번역할 수 있습니다.

### 상품명 번역

- 검색 결과 카드: 상품명 옆의 **번역 아이콘**(🌐)을 클릭하면 바로 아래에 한국어 번역이 표시됩니다.
- 상품 상세 페이지: 상품명 옆의 **번역 아이콘**을 클릭합니다.

### 상품 설명 번역

상품 상세 페이지에서 설명(Description) 섹션 옆의 **번역 아이콘**을 클릭하면 설명 전체가 한국어로 번역됩니다.

### 번역 캐시

번역 결과는 3단계로 캐시됩니다:
1. **메모리 캐시**: 앱 실행 중 동일 텍스트 즉시 반환
2. **SQLite 캐시**: 앱 재실행 후에도 이전 번역 결과 유지
3. **Lingva API**: 캐시에 없는 경우에만 외부 번역 API 호출

> 번역 기능은 인터넷 연결이 필요합니다 (첫 번역 시). 한 번 번역된 텍스트는 로컬에 저장되어 오프라인에서도 확인 가능합니다.

## 5. 즐겨찾기

### 즐겨찾기 추가

검색 결과의 각 카드에 있는 **하트 아이콘**을 클릭하면 즐겨찾기에 추가됩니다.

- 빈 하트: 즐겨찾기 미등록
- 빨간 하트: 즐겨찾기 등록됨

상품 상세 페이지에서도 동일하게 하트 아이콘으로 즐겨찾기를 관리할 수 있습니다.

<!-- [스크린샷: 카드의 하트 아이콘 — 활성/비활성 상태] -->

### 즐겨찾기 목록 보기

1. 사이드바에서 **즐겨찾기**를 클릭합니다.
2. 저장된 모든 즐겨찾기가 카드 그리드로 표시됩니다.

### 즐겨찾기 검색

즐겨찾기 페이지 상단의 검색창에 키워드를 입력하면 이름, 샵 이름, 카테고리 기준으로 필터링됩니다.

### 즐겨찾기 제거

즐겨찾기 목록에서 카드의 **휴지통 아이콘**을 클릭하면 즐겨찾기에서 제거됩니다.

## 6. 컬렉션 관리

컬렉션은 즐겨찾기를 그룹별로 분류하는 폴더 기능입니다.

### 컬렉션 만들기

1. 즐겨찾기 페이지 좌측의 컬렉션 사이드바에서 **"새 컬렉션"**을 클릭합니다.
2. 컬렉션 이름을 입력합니다 (최대 200자).
3. 색상을 선택합니다 (8가지 색상 중 택 1).
4. **"생성"**을 클릭합니다.

### 컬렉션에 상품 추가/제거

1. 즐겨찾기 카드의 **폴더+ 아이콘**을 클릭합니다.
2. 드롭다운에서 원하는 컬렉션을 클릭하여 추가합니다.
3. 이미 추가된 컬렉션은 체크 표시가 나타나며, 다시 클릭하면 제거됩니다.

### 컬렉션별 보기

컬렉션 사이드바에서 컬렉션을 클릭하면 해당 컬렉션에 속한 상품만 표시됩니다.
**"전체"**를 클릭하면 모든 즐겨찾기가 다시 표시됩니다.

### 컬렉션 수정/삭제

컬렉션 이름에 마우스를 올리면 **연필 아이콘**(이름 변경)과 **휴지통 아이콘**(삭제)이 나타납니다.

- **이름 변경**: 연필 아이콘 클릭 > 새 이름 입력 > `Enter` 또는 체크 아이콘
- **삭제**: 휴지통 아이콘 클릭 (컬렉션만 삭제되며, 즐겨찾기 자체는 유지됩니다)

## 7. 태그 관리

태그는 즐겨찾기 상품에 사용자 정의 라벨을 붙이는 기능입니다.

### 태그 추가

1. 즐겨찾기 카드 하단의 **"태그 추가..."** 입력란을 클릭합니다.
2. 태그를 입력하고 `Enter`를 누릅니다.
3. 이전에 사용한 태그가 자동완성 드롭다운에 제안됩니다.

### 태그 제거

태그 옆의 **X** 아이콘을 클릭하면 해당 태그가 제거됩니다.

### 태그 활용

- 태그는 최대 100자까지 입력 가능합니다.
- 하나의 상품에 여러 태그를 추가할 수 있습니다.
- 통계 페이지에서 자주 사용한 태그를 확인할 수 있습니다.

## 9. FAQ / 문제 해결

### Q: 언어를 영어로 바꾸려면?

**A:** 사이드바 하단의 언어 선택에서 **"English"**를 선택하면 전체 UI가 영어로 전환됩니다. 인기 아바타 이름도 영어로 표시됩니다.

### Q: 영어로 검색할 수 있나요?

**A:** 네. 영어를 입력하면 자동으로 일본어 변환 제안이 표시됩니다. 예를 들어 "dress"를 입력하면 "ワンピース"가 제안됩니다. 제안을 클릭하면 해당 일본어로 Booth.pm을 검색합니다.

### Q: 번역 기능은 어떻게 사용하나요?

**A:** 상품 카드나 상세 페이지에서 번역 아이콘(🌐)을 클릭하면 상품명이 한국어로 번역됩니다. 상세 페이지에서는 상품 설명도 번역할 수 있습니다. 번역 결과는 로컬에 캐시되어 같은 텍스트는 즉시 표시됩니다.

### Q: 검색 결과가 나오지 않아요.

**A:** 다음을 확인하세요:

- 인터넷 연결이 되어 있는지 확인합니다.
- 일본어 키워드로 검색하면 결과가 더 정확합니다. 한국어/영어 입력 시 자동 변환 제안을 활용하세요.
- "Rate limited" 오류가 나타나면 잠시 후 다시 시도하세요 (Booth.pm의 요청 제한).

### Q: 좋아요 수가 바로 표시되지 않아요.

**A:** 좋아요(Wish) 데이터는 검색 후 백그라운드에서 순차적으로 불러옵니다. 필터 패널에 "로딩 중..." 표시가 사라지면 모든 데이터가 로드된 것입니다.

### Q: 설치 시 Windows SmartScreen 경고가 나와요.

**A:** 코드 서명이 없는 개인 프로젝트이므로 정상적인 경고입니다. **"추가 정보"** > **"실행"**을 클릭하면 설치가 진행됩니다.

### Q: 즐겨찾기 데이터는 어디에 저장되나요?

**A:** 앱 데이터 폴더의 SQLite 데이터베이스에 로컬 저장됩니다.

- 경로: `C:\Users\{사용자}\AppData\Roaming\com.boothhunter.desktop\boothhunter.db`
- 서버 전송 없이 100% 로컬에서 관리됩니다.

### Q: 컬렉션을 삭제하면 즐겨찾기도 삭제되나요?

**A:** 아니요. 컬렉션을 삭제해도 즐겨찾기 자체는 그대로 유지됩니다. 컬렉션-상품 간의 연결만 제거됩니다.

### Q: 검색이 느려요.

**A:** Booth.pm의 요청 제한 정책에 맞춰 초당 1회로 제한하고 있습니다. 이는 IP 차단을 방지하기 위한 것이며, 정상 동작입니다.

### Q: 앱을 제거하려면?

**A:** Windows 설정 > 앱 > **BoothHunter** > **제거**를 클릭합니다. Linux에서는 패키지 관리자로 제거합니다. 데이터베이스 파일을 완전히 삭제하려면 위의 AppData/data 경로에서 `com.boothhunter.desktop` 폴더를 수동 삭제하세요.

### Q: 이전 버전에서 업데이트하면 데이터가 유지되나요?

**A:** 네. 데이터베이스 마이그레이션이 자동으로 실행되어 기존 즐겨찾기, 컬렉션, 태그, 검색 기록이 모두 유지됩니다.
